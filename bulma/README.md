## float

`is-clearfix` -> 清除浮动

`is-pulled-left` -> 把一个元素放在左边

`is-pulled-right` -> 把一个元素放在右边

## Spacing

`is-marginless` 移除margin

`is-paddingless	` 移除padding

## container

`container`可以在任何上下文中使用，但主要是作为以下任何一种的直接子类:

* `.navbar`
* `.hero`
* `.section`
* `.footer`
